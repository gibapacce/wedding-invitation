# AGENTS.md

## Objetivo

Construir um convite digital interativo de casamento premium.

Toda implementação deve seguir obrigatoriamente:

- SPEC.md
- ARCHITECTURE.md
- UI-UX-FLOW.md
- DESIGN_SYSTEM.md
- TASK_BREAKDOWN.md

## Regras

- Mobile First obrigatório
- Não utilizar backend
- Não utilizar banco de dados
- Não utilizar Supabase
- Não utilizar Redux
- Utilizar Next.js 15 App Router
- Utilizar TypeScript
- Utilizar Tailwind CSS
- Utilizar Server Components sempre que possível
- Utilizar Client Components apenas quando necessário

## Bibliotecas Permitidas

- Framer Motion
- GSAP
- SwiperJS
- React PageFlip
- Lucide React

## Qualidade

- Lighthouse >= 90
- Accessibility >= 95
- SEO >= 95

## Fluxo de Trabalho

Ao iniciar qualquer tarefa:

1. Ler SPEC.md
2. Ler ARCHITECTURE.md
3. Ler DESIGN_SYSTEM.md
4. Ler UI-UX-FLOW.md
5. Executar apenas uma tarefa do TASK_BREAKDOWN.md por vez
6. Atualizar TASK_BREAKDOWN.md ao finalizar
